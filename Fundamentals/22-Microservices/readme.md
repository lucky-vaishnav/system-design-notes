# Microservices Topics

```
1. Microservices Communication   

2. gRPC

3. Circuit Breaker

4. Bulkhead

5. Service Mesh

6. Saga (Microservices Perspective)

7. Outbox Pattern in Microservices
```

**Why this order?**

- Now we'll first learn **how services communicate**.
- Then we'll learn **how communication becomes faster (gRPC)**.
- Then **how communication fails (Circuit Breaker)**.
- Then **how to isolate failures (Bulkhead)**.
- Then **how large companies manage communication (Service Mesh)**.
- Finally we'll revisit **Saga** and **Outbox**, but this time specifically in a microservices architecture.
